рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃЊрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃЏрЃћрЃўрЃџрЃўрЃЌ/рЃърЃљрЃарЃЮрЃџрЃўрЃЌ, рЃерЃћрЃЏрЃЊрЃћрЃњ рЃњрЃљрЃЏрЃЮрЃЊрЃўрЃА, рЃЊрЃљ рЃарЃЮрЃфрЃљ рЃўрЃАрЃћрЃЋ рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃерЃћрЃАрЃЋрЃџрЃљрЃА рЃўрЃњрЃўрЃЋрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃЌ, рЃљрЃдрЃљрЃа рЃБрЃерЃЋрЃћрЃЉрЃА (рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ "рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃърЃљрЃарЃЮрЃџрЃў рЃљрЃю рЃЏрЃћрЃўрЃџрЃўрЃљ").

## рЃерЃћрЃАрЃљрЃФрЃџрЃЮ рЃЏрЃўрЃќрЃћрЃќрЃћрЃЉрЃў

### 1. Browser Cache/localStorage рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ

- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: Browser-рЃўрЃА cache-рЃўрЃА рЃЊрЃљ localStorage-рЃўрЃА рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
- **рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў**:
  1. F12 Рєњ Console Рєњ localStorage.clear() Рєњ Enter
  2. F12 Рєњ Application Рєњ Storage Рєњ Clear site data
  3. рЃљрЃю рЃАрЃарЃБрЃџрЃўрЃљрЃЊ Ctrl+Shift+Delete Рєњ Clear browsing data

### 2. Session Management рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ

- **рЃбрЃћрЃАрЃбрЃў**: рЃАрЃ«рЃЋрЃљ browser рЃљрЃю incognito mode-рЃерЃў рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: рЃЌрЃБ рЃАрЃ«рЃЋрЃљ browser-рЃерЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА, рЃърЃўрЃарЃЋрЃћрЃџ browser-рЃА cache рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ

### 3. Network/API рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ

- **рЃбрЃћрЃАрЃбрЃў**: F12 Рєњ Network рЃбрЃљрЃЉрЃќрЃћ login рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃўрЃА рЃЊрЃарЃЮрЃА Network requests-рЃўрЃА рЃњрЃљрЃЊрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ
- **рЃФрЃћрЃЉрЃюрЃўрЃџрЃў рЃарЃљ**: 200/401 status code, response error message

### 4. JWT Token рЃЋрЃљрЃЊрЃўрЃА рЃњрЃљрЃАрЃЋрЃџрЃљ

- **рЃЏрЃўрЃќрЃћрЃќрЃў**: рЃФрЃЋрЃћрЃџрЃў token localStorage-рЃерЃў рЃарЃЕрЃћрЃЉрЃљ рЃЊрЃљ рЃљрЃ«рЃљрЃџ login-рЃА рЃ«рЃћрЃџрЃА рЃБрЃерЃџрЃўрЃА
- **рЃњрЃљрЃЏрЃЮрЃАрЃљрЃЋрЃљрЃџрЃў**: localStorage-рЃўрЃА рЃАрЃарЃБрЃџрЃў рЃњрЃљрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћрЃЉрЃљ
